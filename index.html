1. Abstract
Project overview & scope

This project investigates using Li-Fi (Light Fidelity) for secure communication in sensitive areas such as military bases, sea ports, and underwater.

Unlike Wi-Fi, which uses radio waves, Li-Fi uses visible or infrared light and is therefore directional, harder to intercept, and more energy efficient. Using LED transmitters, photodiodes/solar receivers, and mirror-based reflection, the system delivers fast, room-confined or short-range underwater transmissions. Proof-of-concept includes text & voice transfer using an Arduino prototype.

Quick points
  • Directional optical link ‚Üí natural containment
  • Low cost Arduino prototype
  • Works in RF-sensitive areas

5. Conceptual Design
Transmitter, receiver & underwater setup

5.1 Transmitter

  • Arduino encodes text/audio into framed packets.
  • LED is modulated at high speed (beyond human perception).
  • Mirrors deflect beams for non-line-of-sight routing.

5.2 Receiver

  • Photodiode detects fluctuations; TIA amplifier cleans the signal.
  • Arduino decodes packets back to text/audio.

5.3 Underwater Setup

  • Use IP68 enclosures, anti-reflective windows and mirrors for underwater paths.
